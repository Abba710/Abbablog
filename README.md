# Abba blog

My personal blog about **indie hacking, building products, technology, self improvement, and the journey of creating things that genuinely solve real problems**.

Here I share my development experience, write about the products I'm building, publish technical articles, and document everything I learn while turning ideas into real software.

This project is built on top of the **Firefly** template, which provided an excellent foundation. Over time, it has been heavily customized and extended to fit my own needs.

## What's Changed

Compared to the original Firefly template, this project includes:

* A custom Projects page
* Custom project cards
* Affiliate link capture and processing logic
* Additional custom components
* New custom pages
* Reworked configuration files
* Customized design and styling
* A completely personalized content structure

The project is actively maintained and continues to evolve.

## Tech Stack

* Astro 7
* TypeScript
* Tailwind CSS 4
* Svelte 5
* Pagefind
* Sharp
* Expressive Code
* Swup
* KaTeX
* Cloudflare
* Biome
* pnpm

## Getting Started

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

The website will be available at:

```text
http://localhost:4321
```

## Build

```bash
pnpm build
```
